# 🌸 ARMONIA - Reiki a Distancia (v1.0.0)

**ARMONIA** es una aplicación de escritorio diseñada para la gestión integral de un Studio Holístico. Esta herramienta permite a las terapeutas (**Luna, Flor y Luciana**) llevar un registro detallado de sus consultantes, sesiones de Reiki y otras terapias, manteniendo la organización y la calidez que el espacio requiere.

---

## ✨ Características Principales

### 📁 Gestión de Consultantes
* **Ficha Técnica Detallada:** Registro de datos personales, motivo de consulta, observaciones públicas y notas privadas.
* **Buscador Inteligente:** Filtrado en tiempo real para encontrar consultantes rápidamente por nombre.
* **Conexión Directa:** Botón de **WhatsApp** integrado para contactar a la clienta con un solo clic sin necesidad de agendarla en el teléfono personal.

### ✨ Registro de Sesiones
* Historial completo de terapias (Reiki, Tarot, Flores de Bach, etc.).
* Control de **Próxima Cita** para seguimiento de tratamientos.
* Interfaz intuitiva con colores relajantes (Lila, Verde Menta y Rosa Viejo).

### 📜 Historial e Impresión
* Visualización de todo el camino recorrido por el consultante.
* **Generación de PDF:** Exportación de fichas completas para entregar o archivar digitalmente.

### 🔐 Seguridad y Respaldo
* **Acceso Protegido:** Sistema de Login para cada terapeuta.
* **Backup Local:** Función de descarga de respaldo en formato `.txt` en la carpeta de Documentos para emergencias.
* **Base de Datos:** Estructura preparada para escalabilidad.

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Interfaz Gráfica:** `CustomTkinter` (Diseño moderno y adaptativo).
* **Base de Datos:** SQL (Estructura relacional).
* **Reportes:** `FPDF` para la creación de documentos PDF.
* **Navegación:** `Webbrowser` para la integración con WhatsApp Web.
